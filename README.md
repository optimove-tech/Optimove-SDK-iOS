## Setup

### CocoaPods

```ruby
platform :ios, '10.0'
use_frameworks!

target '<APPLICATION_TARGET_NAME>' do
    pod 'OptimoveSDK', '~> 2.0'
end

target '<NOTIFICATION_SERVICE_EXTENSION_TARGET_NAME>' do
    pod 'OptimoveNotificationServiceExtension', '~> 2.0'
end
```
